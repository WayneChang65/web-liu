功能說明

快捷鍵

  • 切換輸入法: Ctrl + p
  • 粗體: Ctrl + b
  • 斜體: Ctrl + i
  • 底線: Ctrl + u
  • 放大選取文字: Ctrl + ], Ctrl + Shift + >, 或 Ctrl + 0
  • 縮小選取文字: Ctrl + [, Ctrl + Shift + <, 或 Ctrl + 9

匯出功能

點擊「匯出」按鈕可以將編輯區的內容轉換為 Markdown 格式並下載。此功能會保留以下格式：

  • 粗體
  • 斜體
  • 底線 (以 HTML <u> 標籤保留)
  • 自訂字級大小 (以 HTML <span> 標籤保留)

暫存功能

  • 自動儲存: 您輸入的內容會自動儲存於瀏覽器中。
  • 讀回暫存: 當編輯區為空時，可以點擊「讀回暫存」按鈕來恢復上次的編輯內容。

多文件編輯

  • 點擊上方的 1, 2, 3 數字按鈕，可以在三個獨立的編輯區之間切換。
  • 每個編輯區都有各自獨立的內容與暫存功能，互不干擾。
